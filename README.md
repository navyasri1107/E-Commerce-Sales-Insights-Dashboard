#  E-Commerce Sales Insights Dashboard

##  Project Overview

The E-Commerce Sales Insights Dashboard is a full-stack data analytics web application designed to transform raw e-commerce transaction data into meaningful business insights.
It helps management and stakeholders analyze sales performance using interactive KPIs, charts, date filters, and downloadable reports.

This project simulates a real-world Business Intelligence (BI) workflow, covering data analysis, backend API development, frontend visualization, and deployment.

##  Key Features

 **Interactive Dashboard**

  * Total Sales
  * Total Orders
  * Average Order Value
  * Unique Customers

   **Dynamic Visualizations (Chart.js)**

  * Sales by Month (Line chart with shaded region)
  * Top Products by Sales (Bar chart)
  * Top Categories by Sales (Doughnut chart)

   **Date Range Filtering**

  * All KPIs and charts update based on selected start and end dates

   **Report Generation**

  * Download filtered sales data as a CSV file
